Módulo Homepage

Gestão completa da página inicial do sistema, banners, carousel, conteúdo principal e portal público de transparência. Personalização completa da interface pública.

Sobre o Módulo

O módulo Homepage gerencia a página inicial do sistema e o portal público de transparência. O sistema permite personalização completa da interface pública através de configurações dinâmicas, gestão de carousel de imagens e conteúdo principal.

O módulo inclui páginas legais (privacidade, termos, sobre), portal de transparência com dados públicos de infraestrutura, e portal do agricultor (quando o módulo ProgramasAgricultura está habilitado).

Versão

Categoria

Interface e Portal

Funcionalidades Principais

Recursos completos para gestão da página inicial e portal público

Página Inicial Personalizável

Página inicial completamente configurável através de SystemConfig, permitindo ativar/desativar seções, personalizar textos e conteúdo.

Gestão de Carousel

Sistema completo de carousel de imagens com gestão de slides, ordenação personalizada, links, textos e controle de ativação/desativação.

Configurações Dinâmicas

Sistema de configurações dinâmicas através de SystemConfig, permitindo personalização de títulos, subtítulos, seções, contatos e navbar sem alterar código.

Portal de Transparência

Portal público de transparência com dados agregados de infraestrutura (poços, pontos de água, pontos de luz, estradas) e estatísticas públicas (LGPD compliant).

Visualização por Localidade

Visualização detalhada de infraestrutura por localidade, permitindo consulta pública de dados geográficos e estatísticas locais.

APIs Públicas

APIs públicas para consumo de dados de infraestrutura e estatísticas, permitindo integração com sistemas externos e mapas interativos.

Portal do Agricultor

Portal do agricultor com consulta de programas, eventos, calendário e informações sobre políticas públicas rurais (quando módulo ProgramasAgricultura está habilitado).

Páginas Legais

Páginas legais integradas: privacidade, termos de uso, sobre e desenvolvedor, permitindo transparência e conformidade legal.

Conformidade LGPD

Portal de transparência desenvolvido com conformidade LGPD, exibindo apenas dados agregados e públicos, sem informações pessoais sensíveis.

Estrutura de Dados

O módulo utiliza as tabelas carousel_slides e system_configs para armazenar configurações e conteúdo do carousel.

Tabela: carousel_slides

Campo Tipo Descrição
id bigint Identificador único
title text (nullable) Título do slide (aceita HTML formatado)
description text (nullable) Descrição do slide (aceita HTML formatado)
image string (nullable) Caminho da imagem do slide
link string (nullable) URL de link do slide
link_text string (nullable) Texto do botão de link
order integer Ordem de exibição do slide (padrão: 0)
is_active boolean Indica se o slide está ativo (padrão: true)
show_image boolean Indica se deve exibir a imagem (padrão: true)

Tabela: system_configs

Campo Tipo Descrição
id bigint Identificador único
key string Chave da configuração (ex: 'homepage_hero_title', 'carousel_enabled')
value text Valor da configuração (pode ser string, JSON, boolean, integer)
type string Tipo: 'string', 'integer', 'boolean' ou 'json'
group string Grupo da configuração (ex: 'homepage', 'general')
description text (nullable) Descrição da configuração
is_public boolean Indica se a configuração é pública (acessível via API)

Relacionamentos

O módulo Homepage utiliza modelos do sistema principal (CarouselSlide e SystemConfig) e integra-se com múltiplos módulos para exibir dados públicos de infraestrutura.

Integração com Módulos

O módulo Homepage integra-se com os seguintes módulos para exibir dados públicos no portal de transparência:

  • Localidades: Exibe localidades ativas e suas informações geográficas
  • Poços: Exibe dados públicos de poços (status, localização, vazão)
  • Água: Exibe pontos de distribuição de água (status, conexões, localização)
  • Iluminação: Exibe pontos de luz (status, tipo de lâmpada, potência)
  • Estradas: Exibe trechos de estradas (extensão, pavimentação, condição)
  • Demandas: Exibe estatísticas agregadas de demandas (abertas, concluídas)
  • ProgramasAgricultura: Portal do agricultor com programas, eventos e calendário (quando habilitado)

Métodos e Scopes

Os modelos CarouselSlide e SystemConfig possuem métodos auxiliares e scopes que facilitam consultas e gerenciamento de configurações.

Scopes (CarouselSlide)

  • scopeActive() - Filtra slides ativos
  • scopeOrdered() - Ordena slides por ordem e data de criação

Métodos Estáticos (SystemConfig)

  • get() - Obtém valor de configuração por chave (com conversão de tipo)
  • set() - Define valor de configuração (cria ou atualiza)
  • getByGroup() - Obtém todas as configurações de um grupo

Rotas Públicas

  • GET / - Página inicial
  • GET /privacidade - Página de privacidade
  • GET /termos - Termos de uso
  • GET /sobre - Página sobre
  • GET /desenvolvedor - Página do desenvolvedor
  • GET /portal - Portal de transparência
  • GET /portal/localidade/{id} - Detalhes de localidade
  • GET /portal/api/infraestrutura - API de dados de infraestrutura
  • GET /portal/api/estatisticas - API de estatísticas públicas
  • GET /portal-agricultor - Portal do agricultor (se ProgramasAgricultura habilitado)
