API Reference

Documentação técnica completa dos endpoints REST do sistema. Todas as rotas (exceto Login e Webhooks) exigem autenticação via Bearer Token.

Base URL: https://seu-dominio.com/api/v1

Autenticação

POST /api/login

Autentica usuário e retorna Bearer Token.

Notificações

GET /api/notificacoes

Lista todas as notificações do usuário.

GET /api/notificacoes/unread

Lista apenas notificações não lidas.

GET /api/notificacoes/count

Retorna contagem de não lidas.

POST /api/notificacoes/{id}/read

Marca uma notificação específica como lida.

Chat

GET /api/chat/sessions

Retorna sessões ativas de chat.

GET /api/chat/session/{id}/messages

Histórico de mensagens de uma sessão.

POST /api/chat/session/{id}/message

Envia uma nova mensagem.

Módulo Água

GET /api/v1/agua

Listagem de registros de água (paginado).

POST /api/v1/agua

Criar novo registro.

PUT /api/v1/agua/{id}

Atualizar registro existente.

Módulo Poços

GET /api/v1/pocos

Listagem de poços artesianos.

POST /api/v1/pocos

Cadastrar novo poço.

Módulo Iluminação

GET /api/v1/iluminacao

Listagem de pontos de iluminação.

Módulo Estradas

GET /api/v1/estradas

Monitoramento de estradas e vias.

Módulo Localidades

GET /api/v1/localidades

Cadastro de bairros e distritos.

Módulo Funcionários

GET /api/v1/funcionarios

Gestão de servidores municipais.

Módulo Pessoas (CadÚnico)

GET /api/v1/pessoas

Listagem de beneficiários e cidadãos.

Módulo Avisos

GET /api/v1/avisos

Quadro de avisos e comunicados.

Módulo Blog

GET /api/v1/blogs

Gerenciamento de posts e notícias.

Pagamentos Split (Efipay)

POST /api/pix/cobrar

Gera cobrança PIX com divisão automática de valores.

POST /api/webhook/pix

Callback público para confirmação de pagamento.

Relatórios & Analytics

GET /relatorios/api/demandas

Dados estatísticos de demandas.

GET /relatorios/api/ordens

Dados estatísticos de ordens de serviço.

GET /relatorios/api/materiais

Dados estatísticos de consumo de materiais.
