Módulo de Infraestrutura

Módulo Água

Gestão completa de sistemas de abastecimento de água do município, incluindo redes de distribuição, pontos de abastecimento e controle de qualidade da infraestrutura hídrica.

Sobre o Módulo

O módulo Água é responsável pela gestão completa dos sistemas de abastecimento de água do município. Ele permite o cadastro e controle de redes de distribuição, pontos de abastecimento, manutenção preventiva e integração completa com o sistema de demandas e ordens de serviço.

Versão

1.0.0

Categoria

Infraestrutura Hídrica

Funcionalidades Principais

Recursos completos para gestão de infraestrutura hídrica municipal

Cadastro de Redes

Cadastro completo de redes de distribuição de água com informações técnicas detalhadas (diâmetro, material, extensão, tipo de rede).

Pontos de Distribuição

Gestão de pontos de distribuição (hidrantes, caixas d'água, reservatórios) com localização geográfica e capacidade.

Controle de Status

Acompanhamento do status das redes (funcionando, com vazamento, interrompida) e pontos (funcionando, com defeito, em manutenção).

Integração com Demandas

Integração completa com o módulo de Demandas, permitindo rastreamento de solicitações relacionadas a problemas de água na mesma localidade.

Ordens de Serviço

Rastreamento de ordens de serviço relacionadas através das demandas, permitindo acompanhamento completo da execução de manutenções.

Estatísticas Completas

Dashboard administrativo com estatísticas detalhadas de redes, pontos, demandas e ordens de serviço relacionadas.

Estrutura de Dados

O módulo utiliza duas tabelas principais para gerenciar a infraestrutura hídrica do município.

Tabela: redes_agua

Campo Tipo Descrição
id bigint Identificador único
codigo string (unique) Código único da rede
localidade_id foreignId Localidade onde a rede está localizada
tipo_rede enum principal, secundaria, ramal
diametro string Diâmetro da tubulação (polegadas ou mm)
material string Material da tubulação (PVC, ferro, polietileno)
extensao_metros decimal(10,2) Extensão total da rede em metros
data_instalacao date Data de instalação da rede
status enum funcionando, com_vazamento, interrompida
observacoes text Observações e notas adicionais

Tabela: pontos_distribuicao

Campo Tipo Descrição
id bigint Identificador único
codigo string (unique) Código único do ponto
rede_agua_id foreignId Rede de água à qual pertence
localidade_id foreignId Localidade onde está localizado
endereco string Endereço do ponto de distribuição
latitude / longitude decimal(10,8) Coordenadas geográficas
nome_mapa string Nome para identificação no mapa
numero_conexoes integer Número de conexões do ponto
tipo enum hidrante, caixa_dagua, reservatorio
capacidade_litros decimal Capacidade em litros
status enum funcionando, com_defeito, manutencao

Relacionamentos

O módulo Água mantém relacionamentos integrados com outros módulos do sistema, permitindo rastreamento completo do fluxo de trabalho.

RedeAgua

  • localidade(): belongsTo - Rede pertence a uma localidade
  • pontosDistribuicao(): hasMany - Rede tem muitos pontos de distribuição
  • demandas(): hasMany - Demandas de água na mesma localidade (tipo 'agua')
  • ordensServico(): hasManyThrough - Ordens de serviço através das demandas

PontoDistribuicao

  • redeAgua(): belongsTo - Ponto pertence a uma rede de água
  • localidade(): belongsTo - Ponto pertence a uma localidade
  • demandas(): hasMany - Demandas de água na mesma localidade
  • ordensServico(): hasManyThrough - Ordens de serviço através das demandas

Controllers e Rotas

AguaController

Controller público para operações CRUD de redes de água.

Namespace: Modules\Agua\App\Http\Controllers

Traits: ExportsData, ChecksModuleEnabled

Método Rota Descrição
index() GET /agua Lista todas as redes com filtros e busca
create() GET /agua/create Formulário de criação
store() POST /agua Salva nova rede
show() GET /agua/{id} Exibe detalhes da rede
edit() GET /agua/{id}/edit Formulário de edição
update() PUT/PATCH /agua/{id} Atualiza rede existente
destroy() DELETE /agua/{id} Remove rede (soft delete)
export() GET /agua?format=excel/pdf Exporta dados em Excel ou PDF

AguaAdminController

Controller administrativo com estatísticas completas e visualização detalhada.

Namespace: Modules\Agua\App\Http\Controllers\Admin

Método Rota Descrição
index() GET /admin/agua Lista redes com estatísticas gerais
show() GET /admin/agua/{id} Detalhes completos com relacionamentos
